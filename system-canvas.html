    Offcanvas / Drawer Library

    Standard patterns: filters, details, quick edit, bulk actions.

    Canvas presets

    Use Bootstrap offcanvas with consistent header/actions.

    Backend rule:

    Offcanvas is preferred for non-destructive flows (filters/details). Destructive actions stay in SweetAlert2 confirm.

    JS usage

    Open/close programmatically.

    bootstrap.Offcanvas.getOrCreateInstance('#ocFilters').show()

    bootstrap.Offcanvas.getInstance(el).hide()
